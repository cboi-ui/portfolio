# Pricing Guide

## Public pricing (shown on the site)

Keep it to starting prices only. Never publish the granular/per-frame
breakdown below — it invites "6 frames = $60" scope arguments. Quote actual
jobs using the internal guide further down, in DMs.

| Tier | Starting price |
|---|---|
| Simple UI | $5+ |
| Standard UI | $25+ |
| Complete UI packages | $100+ |

---

## Internal quoting reference (not published)

Pricing depends on complexity, number of elements, required polish,
implementation needs, and revisions — not raw quantity of frames.

### Basic UI — $5–$25
Simple buttons, small popups, basic notifications, counters, small panels.
Includes: UI design, basic assets, one revision.

### Standard UI — $25–$75
Shop screens, daily rewards, settings, leaderboards, basic inventories.
Includes: custom layout, icons/assets, polished design, mobile
considerations.

### Advanced UI — $75–$200+
Full inventory systems, crate systems, simulator UI packages, multi-screen
flows, complete HUD designs.
Includes: full design system, multiple components, consistent styling,
extensive polish.

### Roblox Studio implementation (optional add-on)
Priced separately based on: UI scripting, animations, responsiveness,
integration with existing systems.

### Custom/large projects
Individually quoted — do not force these into the tiers above.

### Per-element sanity check (for quoting only)
- Simple element (button, icon, badge, popup): $5–15 / 1,500–5,000 Robux
- Medium frame (shop panel, settings, daily reward, leaderboard): $20–50 /
  5,000–15,000 Robux
- Complex system (inventory, crates, summon, simulator HUD, full shop):
  $50–150+ / 15,000–40,000+ Robux

Remember: a single complex inventory frame can take longer than five simple
buttons combined. Price the work, not the frame count.

## Payment

Robux or USD, depending on payment method availability. Terms are agreed
before work starts (see `COMMISSION_POLICY.md`).
