# CBoi — Roblox UI Commission Portfolio Website

## Purpose

A professional portfolio/landing page for CBoi's Roblox UI commission work.
Goal: showcase design skill, attract Roblox developers who need UI work, and
make it easy for them to understand pricing and get in touch.

## Target Audience

- Roblox developers and solo devs
- Studio/team projects needing a UI artist
- Simulator, tycoon, and horror-game creators
- Indie Roblox creators commissioning one-off UI pieces

## Main Goals

1. Showcase previous UI work convincingly
2. Make contacting for a commission effortless
3. Set clear pricing expectations without over-committing to a number
4. Build trust (professionalism, consistency, clear process)
5. Convert visitors into commission inquiries on Discord

## Site Requirements

- Hero (name, tagline, immediate visual hook)
- Portfolio gallery (grouped by style/project, not raw image dump)
- Services
- Pricing (starting prices only — see `PRICING_GUIDE.md`)
- Commission process/policy
- Contact (Discord primary)
- About/bio

## Notes

This is a freelance commission landing page first, portfolio gallery second —
every section should nudge toward "contact me on Discord," not just look nice.
